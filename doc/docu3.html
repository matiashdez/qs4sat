get_extended_edges(nodes, edges)
Description:

Generates a set of extended edges by including not just the direct connections (edges), but also the indirect ones formed by the neighbors of neighbors (second-order neighbors). This is useful for enforcing more robust constraints in the graph coloring problem.

Parameters:
  • nodes (int): Total number of nodes in the graph.
  • edges (list[tuple[int, int]]): List of direct connections between nodes.
Returns:

A list of tuples representing both direct and indirect (neighbor-of-neighbor) connections that do not already exist in the original edge list.

How It Works:
  • Constructs an adjacency list.
  • Computes second-degree neighbors.
  • Adds edges between each node and its second-degree neighbors if they are not already present.
Example:

original_edges = [(0, 1), (1, 2)]
extended = get_extended_edges(3, original_edges)
print(extended)  # Might include (0, 2) if it's a second-degree neighbor
